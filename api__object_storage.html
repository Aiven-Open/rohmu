get_or_create_bucket(bucket_name: str) → str

Look up the bucket if it already exists and try to create the bucket in case it doesn’t. Note that we can’t just always try to unconditionally create the bucket as Google imposes a strict rate limit on bucket creation operations, even if it doesn’t result in a new bucket.

Quietly handle the case where the bucket already exists to avoid race conditions. Note that we’ll get a 400 Bad Request response for invalid bucket names (“Invalid bucket name”) as well as for invalid project (“Invalid argument”), try to handle both gracefully.
